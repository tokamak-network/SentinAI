## 핵심 전제

이번 30일은 세 가지에 집중한다.

1. **구독 서비스 기반** — General/Premium 티어를 실제로 판매 가능한 수준으로 구현
2. **운영자 신뢰 확보** — 공개 SLA 페이지, 레퍼런스 케이스, 멀티플랫폼 양방향 승인
3. **영업 파이프라인 구축** — 잠재 고객 직접 발굴 → 외부 체험팀 온보딩 → Aha Moment 설계

> **유료 전환 타임라인:** 30일은 인프라 완성 + 체험 시작이 목표다. 실제 전환은 체험팀이 Aha Moment를 충분히 경험한 60–90일 시점에 자연스럽게 유도한다. 제품이 먼저 증명되어야 한다.

```
  - 승인 대기: 5분 (설정 가능)
  - 타임아웃: 자동 취소 + 전체 채널에 "만료" 편집

```
