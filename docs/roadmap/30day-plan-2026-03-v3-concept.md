## 1. 문서 목적

v2 컨셉은 "운영 가능한 제품 단계로 끌어올리는 것"이 핵심이었다.
v3는 그 다음 질문에 답한다: **운영 가능한 제품을, 어떻게 돈을 버는 사업으로 전환하는가.**

본 문서의 목적은 다음과 같다.

1. v2 → v3 전략 전환의 근거와 논리를 명확히 설명
2. 30일 실행이 **수익 창출 구조**를 만드는 과정임을 제시
3. 기술 구현이 시장 포지셔닝과 고객 획득에 어떻게 연결되는지 서술

## 2. 전략 전환 (v2 → v3)

### 2.1 v2의 한계

v2는 전통적 SaaS 구독 모델을 계획했다 — General(무료) / Premium($299) / Enterprise.
코드베이스 감사와 시장 분석을 통해 근본적 약점이 드러났다.

**셀프호스팅 오픈소스 소프트웨어는 기능 제한을 강제할 수 없다.**

- Feature Gate는 `honor system`에 불과하다 — git clone으로 우회 가능
- 구독 티어에 대한 기술적 강제 수단이 없다
- 무료 사용자를 유료로 전환할 설득 구조가 부재하다

### 2.2 v3의 핵심 통찰

> 코드는 복제할 수 있다. 6개월의 운영 경험은 복제할 수 없다.

SentinAI의 진정한 자산은 소프트웨어가 아니라, **에이전트가 축적한 운영 지식**이다.
Thanos Sepolia에서 6개월간 4,656 사이클을 돌며 학습한 47개 패턴 —
이것은 `git clone`으로 가져갈 수 없다.

### 2.3 Agent-for-Hire 모델

소프트웨어를 파는 것이 아니라, **숙련된 에이전트를 고용**하는 모델이다.

```
소프트웨어: 무료, 오픈소스, 셀프호스팅 (변경 없음)
상품:       검증 가능한 실적을 가진 숙련된 에이전트
가격:       경험 기반 등급 + 성과 보너스
```

| 등급     | 경험      | 월간 요금     | 핵심 가치                     |
|----------|-----------|--------------|-------------------------------|
| Trainee  | 0–30일    | 무료          | 무위험 체험                    |
| Junior   | 1–3개월   | $199/체인     | 기본 자동화 + 이상 탐지         |
| Senior   | 3–6개월   | $499/체인     | 완전 자율 + 예측 + RCA         |
| Expert   | 6개월+    | $799/체인     | 크로스체인 지능 + 커스텀 전략    |

이 모델이 작동하는 이유:

1. **오픈소스가 적이 아니라 아군이다** — 코드 투명성이 신뢰를 만들고, 신뢰가 전환을 만든다
2. **경쟁 해자가 데이터에 있다** — 운영 경험은 시간으로만 쌓인다
3. **가격이 가치를 증명한다** — "이 에이전트가 지난달 12건의 인시던트를 자동 해결했습니다"

## 3. 계획의 목적 (Why)

30일 계획의 목적은 Agent-for-Hire 모델을 **기술적으로 실현 가능한 상태**로 만드는 것이다.

구체적으로 세 가지를 동시에 달성한다.

### 3.1 수익 인프라 구축

에이전트의 경험을 **측정 가능하고, 전이 가능하고, 가격 매길 수 있는 자산**으로 변환한다.

- Experience Transfer: 신규 에이전트가 기존 지식으로 빠르게 부트스트랩
- Collective Intelligence: 다수 인스턴스의 패턴을 집단 지식으로 병합
- Pricing Engine: 경험 등급과 성과 실적을 검증 가능한 가격으로 변환

### 3.2 시장 접근성 확보

잠재 고객이 SentinAI를 발견하고, 이해하고, 시작할 수 있는 경로를 만든다.

- Connect Your Node: 5분 안에 에이전트를 실행할 수 있는 설정 마법사
- 공개 에이전트 프로필: 에이전트의 이력서가 마케팅 자산이 된다
- X 런칭 + 아웃리치: 5개 이상 팀에 직접 연락, 2개 이상 트라이얼 시작

### 3.3 운영 신뢰성 완성

프로덕션 환경에서 안정적으로 운영할 수 있는 기반을 마련한다.

- OpenTelemetry: 에이전트 사이클의 병목을 시각화
- 프로덕션 가이드: 배포, 런북, 트러블슈팅 문서 완성
- 멀티플랫폼 알림: Slack/Discord/Telegram 양방향 승인 체계

## 4. 실행 범위 (What)

### 4.1 수익 인프라 (1주차)

에이전트 경험을 상품화하는 핵심 파이프라인을 구축한다.

```
Experience Store → Pattern Extractor → Experience Transfer → Collective Intelligence
     (기록)            (학습)              (전이)                (집단 지능)
                                             ↓
                                      Pricing Engine → Agent Resume → 공개 프로필
                                         (가격)          (증명)        (마케팅)
```

- **Experience Transfer Protocol**: 동일 프로토콜 타입의 패턴을 50% 신뢰도로 전이
- **Collective Intelligence**: 모든 인스턴스의 패턴을 프로토콜별로 병합
- **Pricing Engine**: 등급 기반 월간 요금 + 성과 기반 보너스 계산
- **멀티플랫폼 알림**: Slack/Discord/Telegram 어댑터 + 양방향 승인

### 4.2 제품 UX + 시장 접근 (2주차)

고객이 SentinAI를 처음 접하는 접점을 설계한다.

- **Connect Your Node 마법사**: 순수 클라이언트 사이드 설정 생성기
- **랜딩 페이지 업데이트**: Agent-for-Hire 메시징 반영
- **공개 에이전트 대시보드**: 에이전트 이력서를 웹에서 열람 가능
- **마케팅 콘텐츠**: 레퍼런스 케이스, 런칭 스레드, 인포그래픽

### 4.3 관찰성 + 프로덕션 (3주차)

운영 안정성과 가시성을 확보한다.

- **OpenTelemetry 트레이싱**: 에이전트 사이클 span + LLM 호출 메트릭
- **프로덕션 가이드**: 배포 절차, 런북, 트러블슈팅 (P3 감사 완료)
- **통합 테스트 + 배포**: 전체 검증 매트릭스 실행

### 4.4 런칭 + 아웃리치 (4주차)

시장에 존재감을 알린다.

- **X 계정 런칭**: 발표 + 에이전트 경험 스레드
- **콘텐츠 캘린더**: 8일간 매일 1개 콘텐츠
- **직접 아웃리치**: 5개 이상 팀 연락, 2개 이상 트라이얼 시작

## 5. 최종 산출물 (Deliverables)

30일 종료 시점에 기대하는 최종 산출물은 아래와 같다.

### 5.1 수익 창출 기반

- 에이전트 경험을 측정하는 **Experience Transfer + Collective Intelligence** 시스템
- 경험 등급과 성과를 가격으로 변환하는 **Pricing Engine**
- 에이전트의 이력과 능력을 공개 증명하는 **Agent Resume + 프로필 페이지**

### 5.2 고객 획득 채널

- 5분 안에 에이전트를 실행할 수 있는 **Connect Your Node 마법사**
- Agent-for-Hire 메시징이 반영된 **랜딩 페이지**
- X 계정 + 레퍼런스 케이스 + 5개 이상 팀 아웃리치

### 5.3 운영 완성도

- 에이전트 사이클 병목을 시각화하는 **OpenTelemetry 트레이싱**
- Slack/Discord/Telegram **양방향 승인 알림**
- 배포/런북/트러블슈팅 **프로덕션 가이드**

## 6. 기대 효과

### 6.1 사업 효과

1. **수익 경로 확보**: Trainee → Junior 전환으로 첫 유료 수익 발생 가능
2. **가격 정당성 확립**: 검증 가능한 실적(자동 해결 건수, 업타임)이 가격을 증명
3. **시장 포지셔닝**: "도구"가 아닌 "숙련된 에이전트를 고용하라"는 독자적 포지션

### 6.2 제품 효과

1. **경험 전이로 온보딩 가속**: 신규 에이전트가 집단 지식으로 즉시 부트스트랩
2. **네트워크 효과 기반 마련**: 에이전트가 많을수록 집단 지능이 강해진다
3. **오픈소스 + 유료 공존**: 코드는 무료, 경험은 유료 — 갈등 없는 수익 모델

### 6.3 기술 효과

1. **관찰성 확보**: OpenTelemetry로 에이전트 사이클의 모든 단계를 추적
2. **멀티채널 운영**: 대시보드 외에 Slack/Discord/Telegram으로 운영 가능
3. **프로덕션 준비 완료**: 문서화된 배포 절차와 런북으로 재현 가능한 운영

## 7. 수익 전망

```
Horizon 1 (현재 → 6개월)    Agent-for-Hire 직접 고용       $5K–15K/월
Horizon 2 (6–12개월)         Agent Marketplace 플랫폼       $20K–50K/월
```

Horizon 2 이후는 이 기반 위에 Marketplace와 서드파티 에이전트가 올라간다.

## 8. 결론

v2는 "운영 가능한 제품"을 만들었다. v3는 그 제품 위에 **사업을 올린다**.

SentinAI v3 30일 계획은 다음을 의미한다.

1. **수익 모델의 기술적 실현** — 경험 전이, 집단 지능, 가격 엔진이 "숙련된 에이전트" 가치 제안을 뒷받침
2. **시장 진입 실행** — 랜딩 페이지, 마법사, X 런칭, 직접 아웃리치로 첫 고객을 만든다
3. **오픈소스와 수익의 공존 증명** — 코드를 공개하면서도 경험이라는 해자로 지속 가능한 수익을 창출

핵심 원칙은 단순하다:
**코드는 복제할 수 있다. 경험은 복제할 수 없다. 우리는 경험을 판다.**
